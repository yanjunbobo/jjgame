Content Corrections

Found a wrong stat, outdated build, or broken link? Tell us which game and guide, what the issue is, and if possible what the correct information should be. We correct verified errors quickly.

💡

Game Requests

Want us to cover a game we haven't added yet? Let us know the game name, platform, and what kind of guide you'd want. Popular requests get prioritized — but we can't promise every game will be added.

🤝

Partnerships & Sponsorships

We don't do sponsored guides or paid reviews. If you're a game publisher, community, or tool developer interested in non-paid collaboration, reach out and we'll see if there's a fit.

🐛

Site Issues

Page not loading, broken images, ads causing layout problems — let us know what browser and device you're on. We test across Chrome, Firefox, and Safari on desktop, but mobile edge cases slip through sometimes.
