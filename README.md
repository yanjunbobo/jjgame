# MetaQuestly

MetaQuestly is an original, fully static game guide and strategy tools website for `https://game.xxym8.com/`. It replaces the old static index style with game hubs, structured guide pages, interactive tools, local search, filters, checklists, SEO routes, and GitHub Pages deployment.

MetaQuestly is independent. Not affiliated with or endorsed by any game publisher. All trademarks belong to their respective owners.

## Tech Stack

- Static HTML, CSS, and JavaScript modules
- Local data files in `src/data`
- Node build script, no backend server
- GitHub Pages compatible `dist`
- No paid API dependencies

## Local Development

```bash
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
```

The generated static site is written to `dist`. It includes route HTML files, `404.html`, `sitemap.xml`, `robots.txt`, `feed.xml`, and `CNAME`.

## Deploy to GitHub Pages

This repository includes `.github/workflows/deploy.yml`. Push to `main`, then enable GitHub Pages with GitHub Actions as the source.

For a custom domain:

1. Keep the generated `CNAME` value as `game.xxym8.com`.
2. In GitHub Pages settings, set the custom domain to `game.xxym8.com`.
3. In DNS, add the required `CNAME` or `A` records for GitHub Pages.

## Change the Website Name

Update:

- `src/lib/seo.js`
- `src/index.html`
- `package.json`
- brand text in `src/components/ui.js`

Then run `npm run build`.

## Add a New Game

Add a new row to the `names` array in `src/data/games.js`. Each game should include:

- slug
- name
- genre
- platforms
- tags
- difficulty
- popularity label
- hero color

Guides and most route entries are generated from the game data.

## Add a New Guide

The starter guide matrix is generated in `src/data/guides.js`. Add a new guide kind to `guideKinds`, or add custom guide content by extending the exported `guides` array.

## Add a New Tool

1. Add metadata in `src/data/tools.js`.
2. Add the interface and result renderer in `src/components/tools.js`.
3. Link it from relevant pages in `src/main.js`.

## Replace Advertisement Code

`AdSlot` is implemented in `src/components/ui.js`. It currently renders a placeholder labeled `Advertisement`. Replace the placeholder markup with your ad provider code when ready.

Supported positions include:

- `homepage-after-hero`
- `sidebar`
- `in-article`
- `before-faq`
- `footer`

## Add Affiliate Modules

`AffiliateBox` is implemented in `src/components/ui.js`. Replace the placeholder categories with real product modules after adding proper affiliate disclosures and compliant outbound links.

Current disclosure:

- `feed.xml`

- canonical tags
- Open Graph tags
- Twitter card tags
- structured data for WebSite, Organization, WebApplication, Article, and ItemList where useful

## Route Matrix

The build creates:

- Home page
- Games index
- 40 game hub pages
- 152 starter, topic, and deep guide pages
- Tool index and tool pages
- Tier list and walkthrough aggregation
- Blog index and article pages
- Genre, platform, tag, and collection landing pages
- Basic pages: About, Contact, Privacy, Terms, Disclaimer, Sitemap
